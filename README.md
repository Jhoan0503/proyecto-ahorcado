# Proyecto Ahorcado

## Descripción

Este proyecto consiste en el desarrollo de un juego del ahorcado utilizando el lenguaje de programación Python. El programa selecciona una palabra de forma aleatoria y permite al usuario adivinarla ingresando letras hasta completar la palabra o quedarse sin intentos.

El proyecto fue desarrollado como parte de la asignatura de Lógica de Programación, aplicando conceptos fundamentales de algoritmos, estructuras condicionales, estructuras repetitivas y validación de datos.

## Funcionalidades implementadas

- Selección aleatoria de palabras.
- Control de intentos del jugador.
- Visualización del progreso de la palabra.
- Validación de letras correctas e incorrectas.
- Validación para ingresar una sola letra.
- Detección de letras repetidas.
- Registro de letras utilizadas durante la partida.
- Mensajes de victoria y derrota.
